# Diffusion-Based Reconstruction and Artifact Analysis of Deepfake Images

## Overview

This project investigates the vulnerability of deepfake detection models against **diffusion-based adversarial purification**.
We apply DifFace (diffusion-based face restoration) to FaceForensics++ deepfake images at three strength levels and analyze how this affects both detection performance and visual artifacts.

## Research Questions

1. Does Diffusion purification significantly reduce deepfake-specific artifacts (FFT anomalies, boundary edges)?
2. Does the FaceForensics++ Xception detector show lower confidence on purified deepfakes?
3. Which artifact metric is most sensitive to Diffusion strength?

## Dataset

- **FaceForensics++ (C23)**: 1,000 source videos x 5 deepfake methods (Face2Face, FaceShifter, FaceSwap, NeuralTextures, Deepfakes)
- **DifFace purification**: applied at s = 10 / 50 / 75
- **Total images**: 40,820

## Methods

| Component | Detail |
|-----------|--------|
| Data preprocessing | MTCNN face crop + DifFace purification |
| Deepfake detector | FaceForensics++ official Xception (C23, inference-only) |
| Artifact metrics | FFT Energy, FFT Peak, Boundary Edge, CNN Feature Distance (ResNet18) |
| Analysis | t-test, ANOVA, Logistic Regression, Correlation Analysis, Qualitative Heatmap |

## Key Results

| Level | Fake Detection Rate | FFT Energy Change | CNN Feature Distance |
|-------|-------------------|-------------------|----------------------|
| Original | 61.15% | - | - |
| s_10 | 57.73% | +0.0012 | 0.036 |
| s_50 | 22.00% | -0.086 | 0.059 |
| s_75 | 16.66% | -0.153 | 0.076 |

- Detection rate drops from **61.15% to 16.66%** at s_75
- FFT Energy increases after Diffusion: artifact replacement, not removal
- Boundary Edge: no significant change (t-test p = 0.962)
